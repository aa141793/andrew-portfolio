# Andrew Alexaitis Portfolio

Recruiter-facing portfolio for Andrew Alexaitis, focused on AI-assisted workflow building, technical operations, implementation support, automation, and practical tooling.

## Local Preview

```sh
python3 -m http.server 8080 --directory projects/web/andrew-portfolio
```

Open:

```text
http://localhost:8080/
```

Or double-click:

```text
start-portfolio-preview.command
```

## Edit Content

Most public-facing text lives in `owner-content.js`.

Resume assets live in `assets/resume/`.
